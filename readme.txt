FACE RECOGNITION ATTENDANCE SYSTEM

Python-based face recognition attendance system using OpenCV, face_recognition, and PIL.
It captures real-time video from webcam, recognizes known faces, and logs attendance with timestamps in a CSV file.

FEATURES
--------
- Admin login system for basic security
- Real-time face detection and recognition
- Automatic CSV attendance logging
- Status colors: Green = Recognized, Red = Unrecognized, White = Detecting
- Timestamped records saved daily (e.g., attendance_2025-11-09.csv)
- Easily add or remove known faces via the 'photo_clean' folder
